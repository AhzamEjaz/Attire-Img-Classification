# Attire-Img-Classification
**Description** : A Simple neural network is used to classify the Fashion Attire images using Pytorch. \
**Tools** : Pytorch, Matplotlib, Jupyter Notebook

---
[Hyperparameters]\
**No. of Epochs**  : 100 \
**Batch Size**  : 364 \
**Learning rate**  : 0.001

---
[Results]\
**Loss**  : 0.052 \
**Accuracy**  : 89.08%
